// 01. Overview

Optimal Utilization

In maths, AI should help you understand the method. Use it for hints, practice, checking steps, and quizzing. You should still be able to explain the reasoning yourself.

Target:

// 05. System Anomalies

Hallucinations: False Positives

AI systems can invent steps, definitions, or results that look structurally sound but are logically flawed. Verification is a mandatory phase of calculation.

Protocol 01
Cross-Verify

Substitute back in, estimate, or solve a simpler version to sanity-check.

Protocol 02
Database Match

Compare with your notes. If it uses an unknown method, request the class method.

Protocol 03
Step Inspection

Treat it like marking: one incorrect step breaks the whole solution tree.

// Fail-Safe Query

“Before you answer, list your assumptions. If you’re not sure, say so. Show the method used at school.”
